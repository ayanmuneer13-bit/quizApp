# Quiz App (JavaScript)

A simple, fast quiz application built using HTML, CSS, and vanilla JavaScript.  
Designed to demonstrate core JavaScript logic, DOM manipulation, and state handling.

## Features
- Dynamic question rendering
- Multiple-choice answers
- Score calculation
- Instant feedback
- Clean UI logic separation
- No external libraries

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
